E1 вҖ” Radial sampling / forward model (single echo time)

Source form yбөў = рқ’Ҝбөў S xМ„бөў + Оөбөў (Eq 1)

Standardized \(y_i = \mathcal{T}_i\, S\, \bar{x}_i + \epsilon_i,\quad i=1,\dots,\text{ETL}\)

Meaning вҖ” the measured radial k-space at echo time TEбөў is: take the true TE image xМ„бөў, apply coil sensitivities S, sample its Fourier transform on the radial trajectory of that TE (рқ’Ҝбөў), add noise Оөбөў.

Dimensions вҖ” \(xМ„бөў вҲҲ в„Ӯ^{N}\), \(S : в„Ӯ^{N} вҶ’ в„Ӯ^{NВ·C}\), \(рқ’Ҝбөў : в„Ӯ^{NВ·C} вҶ’ в„Ӯ^{MВ·C}\), \(yбөў вҲҲ в„Ӯ^{MВ·C}\), \(Оөбөў вҲҲ в„Ӯ^{MВ·C}\). (The paper writes \(yбөў вҲҲ в„Ӯ^{MГ—1}\) per coil; multi-coil stacks over C.)

Assumptions вҖ” linear MR signal model; known S; Gaussian(-ish) complex noise; each TE has its own well-distributed radial angles (pseudo-golden-angle) so рқ’Ҝбөў differs per i.

MRI interpretation вҖ” this is the physics: spin density вҶ’ coil weighting вҶ’ k-space encoding вҶ’ sampling. Radial + golden-angle is what makes RADTSE motion-robust and per-TE incoherent.

Optimization interpretation вҖ” defines the data-fidelity operator; reconstruction inverts this map.

DL connection вҖ” this exact operator (and its adjoint) is baked into the data-consistency (DC) layer; the network never has to learn the physics вҖ” it is handed рқ’Ҝбөў, S.

Verification status вҖ” вң… correct as written.

E2 вҖ” Composite (TE-averaged) reconstruction

Source form x_comp = S* рқ’Ҝ*_comp D y (Eq 2)

Standardized \(x_{\text{comp}} = S^{*}\,\mathcal{T}^{*}_{\text{comp}}\, D\, y\)

Meaning вҖ” stack all TEs' k-space into one dataset y, density-compensate (D, a diagonal ramp that up-weights outer k-space to counter radial oversampling of the center), adjoint-NUFFT back to image space, coil-combine (S*). Result: a single high-SNR image at the average TE.

Dimensions вҖ” \(y вҲҲ в„Ӯ^{MВ·ETLВ·C}\), \(D вҲҲ в„қ^{(MВ·ETL)Г—(MВ·ETL)}\) (diag вүҘ 0), \(рқ’Ҝ*_comp : в„Ӯ^{MВ·ETLВ·C}вҶ’в„Ӯ^{NВ·C}\), \(S* : в„Ӯ^{NВ·C}вҶ’в„Ӯ^{N}\), \(x_comp вҲҲ в„Ӯ^{N}\).

Assumptions вҖ” combining across TEs is acceptable for anatomy (contrast is averaged, lost for T2 fitting). D makes рқ’Ҝ*_comp D approximate a true inverse (gridding recon).

MRI interpretation вҖ” the "composite image": more views вҶ’ fewer streaks вҶ’ best anatomical image, at the cost of T2 information. Ramp DCF is the classic filtered-back-projection weighting.

DL connection вҖ” x_comp (or its zero-filled version) is the input to the composite network f_comp.

Verification status вҖ” вң… correct under the density-compensated-adjoint (gridding) convention.

E3 вҖ” Per-TE image reconstruction

Source form xбөў = S* рқ’Ҝ*бөў DМ„ yбөў (Eq 3)

Standardized \(x_i = S^{*}\,\mathcal{T}^{*}_i\, \bar{D}\, y_i\)

Meaning вҖ” reconstruct each TE image separately (DМ„ = single-TE version of D). Each yбөў is highly undersampled (few views/TE), so xбөў is streaky вҖ” this is exactly the problem the network fixes.

Dimensions вҖ” \(yбөў вҲҲ в„Ӯ^{MВ·C}\), \(DМ„ вҲҲ в„қ^{MГ—M}\), \(рқ’Ҝ*бөў : в„Ӯ^{MВ·C}вҶ’в„Ӯ^{NВ·C}\), \(xбөў вҲҲ в„Ӯ^{N}\).

MRI interpretation вҖ” the multicontrast time series \({xбөў}_{i=1}^{ETL}\) that gets voxel-wise fit to a T2 map via dictionary matching (SEPG signal model). Highly aliased when views/TE is small.

DL connection вҖ” {xбөў} are the channels the TE/T2 network cleans up before projection to the subspace.

Verification status вҖ” вң… correct (bar notation = single-TE slice of D, Audit A4).

E4 вҖ” Principal-component (subspace) model

Source form Ux = U S* в„ұ* D y (Eq 4)

Standardized \(U x = U\, S^{*}\,\mathcal{F}^{*}\, D\, y\) вҹ№ PC images = U В· (per-TE recon)

Meaning вҖ” instead of ETL = 32 TE channels, project onto the first P = 4 SVD components of the SEPG signal dictionary. U maps the 32 TE images to 4 PC images that capture 99.96% of signal-decay variance.

Dimensions вҖ” \(U вҲҲ в„Ӯ^{(NВ·P)Г—(NВ·ETL)}\), \(x вҲҲ в„Ӯ^{NВ·ETL}\) (stacked TE images), \(Ux вҲҲ в„Ӯ^{NВ·P}\).

Why it matters вҖ” (1) cuts compute/memory 32вҶ’4; (2) exploits temporal sparsity, exactly as in CS subspace reconstruction (refs 22, 23). This is the CSвҶ’DL bridge in one line: the learned network operates in a compressed temporal subspace motivated by classical CS.

Notation caveat вҖ” here в„ұ is the same NUFFT called рқ’Ҝ in E1вҖ“E3 (Audit A1).

Verification status вҖ” вң… correct; note glyph switch рқ’ҜвҶ’в„ұ.

E5 / E6 вҖ” Unified forward / adjoint notation

Source form x' = A* D y (Eq 5) Е· = A x' (Eq 6)

Standardized \(x' = A^{*} D y\) (image estimate), \(\hat{y} = A x'\) (project back to k-space)

Definitions (from text, p.2477): - Composite images: A := рқ’Ҝ*_comp S, U = I. - PC images: A := в„ұ* S U*, U as in E4. - Convention (A2): thereafter A* := Aбҙҙ D, so A,A* are not true adjoints; D is used only in the image direction (k-spaceвҶ’image), never in the forward (imageвҶ’k-space) direction.

Meaning вҖ” a single symbol pair to describe both pipelines. A* (with D) = go from measured k-space to image; A = go from image to predicted k-space Е·.

Optimization interpretation вҖ” A,A* are the forward/back operators inside every DC step (E7вҖ“E9) and the self-supervised loss (E12вҖ“E13).

DL connection вҖ” the whole point of physics-driven DL: the network is wrapped around fixed A,A*. рҹ”Қ For your code (Audit A2): implement a true Aбҙҙ that passes the dot-product test вҹЁAx, yвҹ© вүҲ вҹЁx, Aбҙҙyвҹ©, and separately a preconditioned A*_paper = Aбҙҙ D. Test the first; use the second in DC.

Verification status вҖ” вң… correct only under the stated non-adjoint convention вҖ” flag when teaching.

E7 вҖ” Data-consistency objective (per cascade)

Source form \(ВҪ вҖ– A_{CNN}^{(j)}(x'^{(j-1)}) вҲ’ y вҖ–ВІ\) (Eq 7)

Standardized \(\tfrac12\big\| A\, f_{\text{CNN}}^{(j)}\!\big(x'^{(j-1)}\big) - y \big\|_2^2\)

Meaning вҖ” after the j-th CNN block denoises the current image, measure how far its predicted k-space A f_CNN(x') is from the acquired k-space y. DC minimizes this.

Dimensions вҖ” \(x'^{(j-1)} вҲҲ в„Ӯ^{NВ·П„}\) (П„ = 1 composite, П„ = P PC), \(y вҲҲ в„Ӯ^{MВ·ETLВ·C}\).

Optimization interpretation вҖ” the quadratic data term of the unrolled variational problem; the CNN supplies the (learned, implicit) regularizer.

Verification status вҖ” вң… correct.

E8 вҖ” Gradient of the DC objective

Source form \(d/dxМӮ'^{(j)} [ ВҪвҖ–AxМӮ'^{(j)} вҲ’ yвҖ–ВІ ] = A*(AxМӮ'^{(j)} вҲ’ y)\) (Eq 8)

Standardized \(\nabla_{\hat{x}'^{(j)}}\Big[\tfrac12\|A\hat{x}'^{(j)}-y\|_2^2\Big] = A^{*}\big(A\hat{x}'^{(j)} - y\big)\)

Derivation вҖ” f(x)=ВҪвҖ–AxвҲ’yвҖ–ВІ. For complex x, the Wirtinger gradient is вҲҮf = Aбҙҙ(AxвҲ’y). The paper writes A* which under convention A2 is AбҙҙD вҹ№ this is a preconditioned gradient, not the exact one.

MRI interpretation вҖ” residual in k-space (AxМӮ'вҲ’y) pushed back to image space and used to correct the image.

DL connection вҖ” this is the gradient the DC layer descends; because О· is learned (E9), backprop can absorb the D preconditioner, so the swap is harmless in practice (Audit A8).

Verification status вҖ” вң… correct as a preconditioned gradient under convention A2. (Exact gradient with a true adjoint would drop D.)

E9 вҖ” Data-consistency update (learned gradient descent)

Source form \(x''^{(j)} = DC^{(j)}(xМӮ'^{(j)}) = xМӮ'^{(j)} вҲ’ О· A*(A xМӮ'^{(j)} вҲ’ y)\) (Eq 9)

Standardized \(x''^{(j)} = \hat{x}'^{(j)} - \eta\, A^{*}\big(A\hat{x}'^{(j)} - y\big)\), with \(xМӮ'^{(j)} := f_{\text{CNN}}^{(j)}(x'^{(j-1)})\), \(x'^{(0)} = A* y\)

Meaning вҖ” one gradient-descent step toward data consistency after each CNN denoise. О· is a trainable scalar step size learned by backprop. K = 5 such CNNвҶ’DC cascades.

This is the heart of the paper. Compare directly to proximal gradient / ISTA:

ISTA:              x^{k+1} = prox_{ОіО»R}( x^k вҲ’ Оі Aбҙҙ(Ax^k вҲ’ y) )
Unrolled (E9):     x^{k+1} = DC^{(k)}( f_CNN^{(k)}(x^k) )     вҶҗ CNN replaces prox, learned О· replaces Оі

The CNN plays the role of the proximal operator of an implicit, learned regularizer; DC enforces the physics.

DL connection вҖ” a textbook model-based DL / unrolled-optimization block (MoDL/VarNet family, refs 25вҖ“30). Weights may be shared or per-cascade; here О· is learned.

Verification status вҖ” вң… correct (learned-step preconditioned gradient descent).

E11 вҖ” Supervised image-space loss (composite)

Source form L_x(xМӮ,x) = Оұ вҖ–xМӮвҲ’xвҖ–вӮҒ/вҖ–xвҖ–вӮҒ + (1вҲ’Оұ) вҖ–xМӮвҲ’xвҖ–вӮӮ/вҖ–xвҖ–вӮӮ (Eq 11)

Standardized \(\mathcal{L}_x(\hat{x},x)=\alpha\dfrac{\|\hat{x}-x\|_1}{\|x\|_1}+(1-\alpha)\dfrac{\|\hat{x}-x\|_2}{\|x\|_2},\quad \alpha=\tfrac12\)

Meaning вҖ” scale-invariant blend of relative в„“В№ (sharp/robust) and relative в„“ВІ (smooth) image error. Normalizing by вҖ–xвҖ– makes the loss invariant to overall image scale.

Statistical note (Expert E) вҖ” mixing в„“В№/в„“ВІ is a robustness/MAP-flavored choice, not an unbiased estimator of anything in particular; treat the 50/50 split as a tuned hyperparameter (Audit A9).

Verification status вҖ” вң… correct.

E12 вҖ” Self-supervised k-space projection (TE/T2 network)

Source form \(Е· = A f_{T2}(y', S)\) (Eq 12)

Standardized \(\hat{y} = A\, f_{T2}(y', S)\)

Meaning вҖ” for T2 mapping there is no clean target (views/TE вүӨ 12, no well-sampled reference). So the network output f_T2(y') is projected back to k-space by A, and the loss is computed in k-space against the acquired data y. This is a self-supervised, physics-consistency loss.

Contrast with SSDU (ref 43) вҖ” vanilla SSDU splits k-space into disjoint input and loss sets. Here the loss is computed over the entire acquired y (not a disjoint held-out subset) "to further enforce data consistency." рҹ”ҙ Common-mistake flag: this weakens the leakage protection SSDU was designed for вҖ” see Audit note and Comparison #18 (supervised vs self-supervised). Worth interrogating for your own work.

Verification status вҖ” вң… correct as stated; вҡ пёҸ the "loss over entire y" choice is a design decision to scrutinize.

E13 вҖ” Self-supervised k-space loss (TE/T2)

Source form L_y(Е·,y) = Оұ вҖ–Е·вҲ’yвҖ–вӮҒ/вҖ–yвҖ–вӮҒ + (1вҲ’Оұ) вҖ–Е·вҲ’yвҖ–вӮӮ/вҖ–yвҖ–вӮӮ (Eq 13)

Standardized \(\mathcal{L}_y(\hat{y},y)=\alpha\dfrac{\|\hat{y}-y\|_1}{\|y\|_1}+(1-\alpha)\dfrac{\|\hat{y}-y\|_2}{\|y\|_2},\quad \alpha=\tfrac12\)

Meaning вҖ” same relative в„“В№/в„“ВІ blend as E11 but in k-space. Minimizing k-space error is the only training signal for the T2 pathway. Note (2.5): T2 error is not in the loss вҖ” it is used only for model selection (pick the epoch with lowest ROI T2 error on the validation set).

Verification status вҖ” вң… correct.
